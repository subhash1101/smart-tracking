## Quick Start (5 Commands)

```bash
# 1. Activate your virtual environment (create it first if needed)
python3 -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate     # Mac/Linux

# 2. Install packages
pip install django scikit-learn pandas numpy

# 3. Train the ML model (creates stress_model.pkl, label_encoder.pkl, scaler.pkl)
python tracker/ml/train_model.py

# 4. Set up database
python manage.py makemigrations tracker
python manage.py migrate

# 5. Run server
python manage.py runserver
```

Open → http://127.0.0.1:8000/

Optional — create admin account:
```bash
python manage.py createsuperuser
# then visit http://127.0.0.1:8000/admin/
```

Run tests:
```bash
python manage.py test tracker
```

## Edit / Update Functionality

The system automatically detects whether a user has already submitted an entry today:

- **No entry today** → Shows a blank form → Saves new `MoodEntry` and `PredictionResult`
- **Entry exists today** → Pre-fills the form with existing data → Shows edit banner → On submit, UPDATES the existing record and re-runs ML prediction

Same URL (`/entry/`) handles both cases — the view detects which mode to use.

## ML Model Details

| Item | Detail |
|------|--------|
| Algorithm | Random Forest (150 trees) |
| Features | 16 input features |
| Training data | 1,500 synthetic samples |
| Target | stress_category (Low/Medium/High) |
| Expected accuracy | ~87–92% |
| Comparison | Logistic Regression vs Decision Tree vs Random Forest |
| Burnout detection | Rule-based score (separate from ML) |
| Depression detection | Count of 4 Boolean indicators (0–4) |
